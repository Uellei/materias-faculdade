ABOUT PROJECT

Instrumentação

Eletrônica

Neste projeto, aplicamos os princípios da instrumentação eletrônica para desenvolver uma estação meteorológica de baixo custo. Utilizamos sensores como BMP280, DHT11, BH1750, AS5600, e um sensor de barreira para medir pressão e temperatura, umidade, luminosidade, ângulo e velocidade/RPM, respectivamente. A aplicação desses sensores permite a captura de dados precisos sobre as condições climáticas, contribuindo para o monitoramento eficiente e econômico.

Sistemas de Controle

a

Nossa estação meteorológica utiliza dados do sensor de temperatura (DHT11) para controlar automaticamente uma cortina no viveiro agrícola em resposta a temperaturas elevadas, otimizando o ambiente de cultivo. Estratégias eficientes, como intervalos de leitura programados, são implementadas para equilibrar precisão e economia de energia. Essa abordagem prática dos Sistemas de Controle na agricultura destaca a utilidade dos dados na gestão térmica do viveiro, promovendo um cultivo mais eficiente e adaptável.

Servidor

a

A estratégia de envio de dados para um servidor remoto via requisições HTTP, implementado em Flask, destaca a importância de um servidor na arquitetura do projeto. Essa abordagem permite o armazenamento persistente dos dados, centralizando as informações de forma segura. A centralização dos dados em um servidor remoto não apenas facilita o monitoramento, análise e visualização, mas também protege contra perda ou danos nos dispositivos locais. A inclusão do servidor como parte integral do sistema contribui para a robustez e confiabilidade da estação meteorológica de baixo custo.
